# 📌 Libreria `csv` in Python

La libreria standard **`csv`** serve per leggere e scrivere file CSV (Comma Separated Values), cioè file di testo strutturati in righe e colonne.

📌 Modulo incluso in Python → **non serve installare nulla**.

```python
import csv
```

---

# 1) 🔹 Apertura corretta dei file CSV

In Python si apre quasi sempre così:

```python
with open("file.csv", mode="r", newline="", encoding="utf-8") as f:
    ...
```

📌 `newline=""` è importante per evitare righe vuote in Windows.

---

# 2) ✅ Lettura CSV base (`csv.reader`)

### File CSV tipo:

```
Mario,Rossi,20
Anna,Bianchi,22
```

### Codice:

```python
import csv

with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.reader(f)

    for riga in reader:
        print(riga)
```

📌 Output:

```
['Mario', 'Rossi', '20']
['Anna', 'Bianchi', '22']
```

---

# 3) Lettura con separatore diverso (`delimiter`)

Molti CSV italiani usano `;`:

```
Mario;Rossi;20
Anna;Bianchi;22
```

```python
with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.reader(f, delimiter=";")

    for riga in reader:
        print(riga)
```

---

# 4) Lettura con intestazione (`csv.DictReader`)

### CSV:

```
nome,cognome,eta
Mario,Rossi,20
Anna,Bianchi,22
```

### Codice:

```python
import csv

with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.DictReader(f)

    for riga in reader:
        print(riga["nome"], riga["cognome"], riga["eta"])
```

📌 Ogni riga è un dizionario:

```python
{'nome': 'Mario', 'cognome': 'Rossi', 'eta': '20'}
```

---

# 5) Leggere tutto in una lista

```python
with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.reader(f)
    dati = list(reader)

print(dati)
```

---

# 6) Saltare intestazione manualmente

Se non vuoi DictReader:

```python
with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.reader(f)

    next(reader)  # salta prima riga (header)

    for riga in reader:
        print(riga)
```

---

# 7) ✅ Scrittura CSV base (`csv.writer`)

```python
import csv

studenti = [
    ["Mario", "Rossi", 20],
    ["Anna", "Bianchi", 22]
]

with open("output.csv", mode="w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)

    for s in studenti:
        writer.writerow(s)
```

📌 File creato:

```
Mario,Rossi,20
Anna,Bianchi,22
```

---

# 8) Scrittura di più righe (`writerows`)

```python
with open("output.csv", mode="w", newline="", encoding="utf-8") as f:

```

# 10) Scrittura con intestazione (`DictWriter`)

```python
import csv

studenti = [
    {"nome": "Mario", "cognome": "Rossi", "eta": 20},
    {"nome": "Anna", "cognome": "Bianchi", "eta": 22}
]

with open("output.csv", mode="w", newline="", encoding="utf-8") as f:
    campi = ["nome", "cognome", "eta"]
    writer = csv.DictWriter(f, fieldnames=campi)

    writer.writeheader()
    writer.writerows(studenti)
```

📌 File:

```
nome,cognome,eta
Mario,Rossi,20
Anna,Bianchi,22
```

# 12) Quote e gestione virgolette

Se i dati contengono virgole:

```
"Rossi, Mario",20
```

Python gestisce automaticamente le virgolette.

### Forzare virgolette sempre:

```python
writer = csv.writer(f, quoting=csv.QUOTE_ALL)
```

### Quote solo se necessario (default):

```python
writer = csv.writer(f, quoting=csv.QUOTE_MINIMAL)
```

Opzioni principali:

* `csv.QUOTE_MINIMAL` (default)
* `csv.QUOTE_ALL`
* `csv.QUOTE_NONNUMERIC`
* `csv.QUOTE_NONE`

---

# 13) Gestione di caratteri speciali (accenti)

Sempre meglio usare:

```python
encoding="utf-8"
```

Se un CSV arriva da Excel Windows, può essere `latin-1`:

```python
encoding="latin-1"
```

---

# 14) Gestione newline e righe vuote (problema comune)

In scrittura usare SEMPRE:

```python
newline=""
```

Altrimenti Windows aggiunge righe vuote.

---

# 15) Conversione dati (numeri e float)

CSV legge sempre stringhe:

```python
eta = int(riga["eta"])
```

Esempio:

```python
with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.DictReader(f)

    for riga in reader:
        riga["eta"] = int(riga["eta"])
        print(riga)
```

---

# 16) Cambiare il carattere di escape

Se usi QUOTE_NONE devi gestire escape:

```python
writer = csv.writer(f, quoting=csv.QUOTE_NONE, escapechar="\\")
```

---

# 17) Parametri importanti di `csv.reader` / `writer`

### Lettura

```python
csv.reader(f,
    delimiter=",",
    quotechar='"',
    escapechar=None,
    skipinitialspace=False
)
```

### Scrittura

```python
csv.writer(f,
    delimiter=",",
    quotechar='"',
    quoting=csv.QUOTE_MINIMAL,
    lineterminator="\n"
)
```

# 18) Dialect (stili predefiniti)

Python permette dialetti:

```python
csv.register_dialect("mio", delimiter=";", quoting=csv.QUOTE_MINIMAL)

with open("file.csv", "r", newline="", encoding="utf-8") as f:
    reader = csv.reader(f, dialect="mio")
```

# 19) Esempio completo: leggere, filtrare, riscrivere

### Input:

```
nome,cognome,eta
Mario,Rossi,20
Anna,Bianchi,22
Luca,Verdi,17
```

### Script:

```python
import csv

studenti_maggiorenni = []

with open("studenti.csv", mode="r", newline="", encoding="utf-8") as f:
    reader = csv.DictReader(f)

    for riga in reader:
        if int(riga["eta"]) >= 18:
            studenti_maggiorenni.append(riga)

with open("maggiorenni.csv", mode="w", newline="", encoding="utf-8") as f:
    campi = ["nome", "cognome", "eta"]
    writer = csv.DictWriter(f, fieldnames=campi)

    writer.writeheader()
    writer.writerows(studenti_maggiorenni)
```

# 20) Lettura CSV da stringa (senza file)

```python
import csv
import io

testo = "nome,cognome\nMario,Rossi\nAnna,Bianchi\n"

f = io.StringIO(testo)
reader = csv.reader(f)

for riga in reader:
    print(riga)
```

---

# 21) Scrittura CSV su stringa (senza file)

```python
import csv
import io

output = io.StringIO()
writer = csv.writer(output)

writer.writerow(["nome", "cognome"])
writer.writerow(["Mario", "Rossi"])

print(output.getvalue())
```

---

# 22) Best practice consigliate

✅ Usa `DictReader` e `DictWriter` se hai intestazioni
✅ Usa `newline=""` sempre
✅ Usa `encoding="utf-8"`
✅ Converti i numeri manualmente (`int()`, `float()`)
✅ Gestisci delimitatori (`;` in Italia è comune)

---

# 23) Errori tipici

### ❌ Righe vuote in output

👉 manca `newline=""`

### ❌ Separatore errato

👉 CSV usa `;` ma stai leggendo con `,`

### ❌ Problemi con caratteri accentati

👉 encoding sbagliato

# 24) Mini-tabella riepilogo

| Funzione               | Uso                          |
| ---------------------- | ---------------------------- |
| `csv.reader()`         | lettura righe come liste     |
| `csv.DictReader()`     | lettura righe come dizionari |
| `csv.writer()`         | scrittura righe da liste     |
| `csv.DictWriter()`     | scrittura righe da dizionari |
| `writer.writerow()`    | scrive una riga              |
| `writer.writerows()`   | scrive più righe             |
| `writer.writeheader()` | scrive intestazione          |
