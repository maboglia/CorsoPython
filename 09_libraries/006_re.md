# 📌 Libreria `re` (Regular Expressions) – Scheda Completa

Serve per lavorare con **pattern di testo**, ricerca e sostituzione avanzata.

```python
import re
```

---

# 1) Funzioni principali di `re`

| Funzione        | Uso                                      |
| --------------- | ---------------------------------------- |
| `re.search()`   | trova la prima occorrenza in una stringa |
| `re.match()`    | match solo all’inizio stringa            |
| `re.findall()`  | lista di tutte le occorrenze             |
| `re.finditer()` | iteratore di match                       |
| `re.sub()`      | sostituzione                             |
| `re.split()`    | split avanzato                           |
| `re.compile()`  | precompila regex                         |

---

# 2) `re.search()` (prima occorrenza)

```python
import re

testo = "Ho 2 cani e 3 gatti"
m = re.search(r"\d+", testo)

print(m.group())   # 2
```

📌 `\d+` = uno o più numeri.

---

# 3) `re.match()` (solo inizio stringa)

```python
re.match(r"\d+", "123abc")   # match
re.match(r"\d+", "abc123")   # None
```

---

# 4) `re.findall()` (tutte le occorrenze)

```python
import re

testo = "Ho 2 cani e 3 gatti"
numeri = re.findall(r"\d+", testo)

print(numeri)  # ['2', '3']
```

---

# 5) `re.finditer()` (match con posizione)

```python
import re

testo = "Ho 2 cani e 3 gatti"

for m in re.finditer(r"\d+", testo):
    print(m.group(), m.start(), m.end())
```

---

# 6) `re.sub()` (sostituzione)

```python
import re

testo = "Ho 2 cani"
nuovo = re.sub(r"\d+", "10", testo)

print(nuovo)  # Ho 10 cani
```

---

# 7) `re.split()` (split avanzato)

```python
import re

testo = "uno,due;tre quattro"
pezzi = re.split(r"[ ,;]+", testo)

print(pezzi)
```

---

# 8) Classi di caratteri più usate

| Pattern | Significato         |
| ------- | ------------------- |
| `\d`    | cifra (0-9)         |
| `\D`    | non cifra           |
| `\w`    | lettera/cifra/_     |
| `\W`    | non alfanumerico    |
| `\s`    | spazio/tab/newline  |
| `\S`    | non spazio          |
| `.`     | qualsiasi carattere |

---

# 9) Quantificatori

| Pattern | Significato   |
| ------- | ------------- |
| `*`     | 0 o più       |
| `+`     | 1 o più       |
| `?`     | 0 o 1         |
| `{n}`   | esattamente n |
| `{n,}`  | almeno n      |
| `{n,m}` | tra n e m     |

Esempio:

```python
re.findall(r"a+", "caaasa")  # ['aaa', 'a']
```

---

# 10) Gruppi `(...)`

```python
import re

testo = "Mario Rossi"
m = re.search(r"(\w+)\s+(\w+)", testo)

print(m.group(1))  # Mario
print(m.group(2))  # Rossi
```

---

# 11) Gruppi con nome

```python
m = re.search(r"(?P<nome>\w+)\s+(?P<cognome>\w+)", "Mario Rossi")
print(m.group("nome"))
print(m.group("cognome"))
```

---

# 12) Alternanza `|` (OR)

```python
re.findall(r"gatto|cane", "cane gatto cane")
```

Output:

```python
['cane', 'gatto', 'cane']
```

---

# 13) Inizio/fine stringa

| Simbolo | Significato    |
| ------- | -------------- |
| `^`     | inizio stringa |
| `$`     | fine stringa   |

Esempio:

```python
re.match(r"^ciao", "ciao mondo")  # match
re.match(r"^ciao", "mondo ciao")  # no
```

---

# 14) Caratteri speciali da escapare

Questi vanno escapati con `\`:

```
. ^ $ * + ? { } [ ] \ | ( )
```

Esempio cercare un punto:

```python
re.search(r"\.", "ciao.txt")
```

---

# 15) Pattern per email (semplice)

```python
import re

email = "mario.rossi@gmail.com"
ok = re.match(r"^[\w\.-]+@[\w\.-]+\.\w+$", email)

print(ok is not None)
```

---

# 16) Pattern per numero di telefono (esempio)

```python
import re

tel = "+39 333-1234567"
ok = re.match(r"^\+?\d{1,3}\s?\d{3}[- ]?\d{7}$", tel)

print(ok is not None)
```

---

# 17) Flags (opzioni)

| Flag            | Significato                  |
| --------------- | ---------------------------- |
| `re.IGNORECASE` | ignora maiuscole/minuscole   |
| `re.MULTILINE`  | ^ e $ lavorano per ogni riga |
| `re.DOTALL`     | `.` include newline          |
| `re.VERBOSE`    | regex leggibile con commenti |

Esempio:

```python
re.findall(r"ciao", "CIAO ciao", flags=re.IGNORECASE)
```

---

# 18) Regex compilata (`re.compile`)

Utile se usi la stessa regex più volte.

```python
import re

pattern = re.compile(r"\d+")

print(pattern.findall("a1 b22 c333"))
```

---

# 19) Esempio pratico: validazione data gg/mm/aaaa

```python
import re

data = "11/04/2026"
ok = re.match(r"^\d{2}/\d{2}/\d{4}$", data)

print(ok is not None)
```

---

# 20) Best practice re

✅ usare `r"..."` (raw string)
✅ usare `compile()` se ripeti molte volte
✅ usare gruppi con nome per parsing complessi
⚠️ regex non valida i significati logici (es 99/99/9999 è sintatticamente ok)
