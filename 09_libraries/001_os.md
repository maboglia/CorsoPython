# Modulo `os`: Il ponte verso il File System

Il modulo `os` permette al codice Python di "guardare fuori" dalla propria finestra e interagire con le cartelle e i file del computer.

* **Concetto chiave:** L'indipendenza dalla piattaforma. Usare `os.path.join("cartella", "file.txt")` garantisce che il codice funzioni sia su Windows (che usa `\`) che su Linux/Mac (che usano `/`).
* **Snippet Didattico:**

```python
import os
# Creare una cartella solo se non esiste
if not os.path.exists("dati_esercitazione"):
    os.mkdir("dati_esercitazione")

```

* **Pro-Tip:** Oggi esiste anche `pathlib` (più moderno e orientato agli oggetti), ma `os` resta lo standard fondamentale da conoscere.

---

```python
import os

# Informazioni directory
print(f"Directory corrente: {os.getcwd()}")
print(f"Contenuto directory: {os.listdir('.')}")

# Manipolazione percorsi
percorso = os.path.join("cartella", "file.txt")
print(f"Percorso completo: {os.path.abspath(percorso)}")

# Variabili d'ambiente
print(f"Utente: {os.getenv('USER', 'Sconosciuto')}")
```

# 📌 Libreria `os` – Scheda Completa

Serve per interagire con **sistema operativo**: file, cartelle, variabili ambiente.

```python
import os
```

---

## 1) Directory corrente

```python
import os

print(os.getcwd())     # current working directory
```

Cambiare directory:

```python
os.chdir("C:/Users/...")
```

---

## 2) Listare file e cartelle

```python
import os

contenuto = os.listdir(".")
print(contenuto)
```

---

## 3) Creare cartelle

```python
import os

os.mkdir("nuova_cartella")           # crea singola cartella
os.makedirs("a/b/c")                 # crea struttura completa
```

---

## 4) Eliminare cartelle

```python
os.rmdir("cartella_vuota")           # solo se vuota
```

---

## 5) Eliminare file

```python
os.remove("file.txt")
```

---

## 6) Rinominare / spostare file

```python
os.rename("vecchio.txt", "nuovo.txt")
```

---

## 7) Verificare esistenza file/cartella

```python
os.path.exists("file.txt")
os.path.isfile("file.txt")
os.path.isdir("cartella")
```

---

## 8) Percorsi con os.path

```python
import os

percorso = os.path.join("cartella", "file.txt")
print(percorso)
```

Separare nome e cartella:

```python
cartella = os.path.dirname(percorso)
nomefile = os.path.basename(percorso)
```

Estensione:

```python
nome, est = os.path.splitext("foto.png")
print(nome, est)   # foto .png
```

---

## 9) Dimensione file

```python
import os

size = os.path.getsize("file.txt")
print(size, "bytes")
```

---

## 10) Variabili ambiente

```python
import os

print(os.environ["PATH"])
```

Oppure più sicuro:

```python
print(os.getenv("PATH"))
```

Impostare variabile ambiente:

```python
os.environ["MIA_VAR"] = "valore"
```

---

## 11) Eseguire comandi di sistema

```python
import os

os.system("dir")   # Windows
os.system("ls")    # Linux/Mac
```

⚠️ `os.system()` è vecchio: meglio `subprocess`.

---

## 12) Best practice

✅ usare `os.path.join()` invece di concatenare stringhe
✅ controllare sempre `exists()` prima di cancellare
⚠️ preferire `pathlib` per progetti moderni
