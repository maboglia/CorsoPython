## **Capitolo: Strutture Dati in Python – Sommario con descrizioni**

1. **Lists**
   Le liste sono collezioni ordinate e mutabili in Python, utilizzate per memorizzare sequenze di elementi eterogenei. Costituiscono una delle strutture dati più versatili del linguaggio.

2. **Accessing Items**
   Python permette di accedere agli elementi delle liste tramite indici, slicing e indici negativi, offrendo un controllo preciso sulla manipolazione dei dati.

3. **List Unpacking**
   Il "unpacking" consente di estrarre più valori da una lista in un’unica istruzione, migliorando leggibilità e compattezza del codice.

4. **Looping over Lists**
   Cicli `for` e altre tecniche permettono di iterare facilmente sugli elementi delle liste, con possibilità di accedere simultaneamente a elementi e indici.

5. **Adding or Removing Items**
   Metodi come `append`, `insert`, `remove`, `pop` e slicing permettono di modificare dinamicamente il contenuto delle liste.

6. **Finding Items**
   È possibile cercare elementi usando operatori come `in`, il metodo `index` e varie tecniche di scansione e filtraggio.

7. **Sorting Lists**
   Python include metodi per ordinare le liste in-place o per restituire nuove liste ordinate, anche tramite funzioni personalizzate.

8. **Lambda Functions**
   Le lambda sono funzioni anonime, spesso usate come callback o come chiavi di ordinamento per funzioni come `sorted` o `map`.

9. **Map Function**
   `map()` applica una funzione a ogni elemento di un iterabile, generando una nuova sequenza trasformata.

10. **Filter Function**
    `filter()` seleziona solo gli elementi che soddisfano una condizione, mantenendo il codice compatto e leggibile.

11. **List Comprehensions**
    Le list comprehension permettono di creare liste in maniera elegante e concisa, integrando trasformazioni e filtri in un’unica espressione.

12. **Zip Function**
    `zip()` consente di unire più sequenze in parallelo, utile per iterare su elementi collegati tra loro.

13. **Stacks**
    Le liste possono essere utilizzate come stack (LIFO) tramite metodi come `append` e `pop`, utili in numerosi algoritmi.

14. **Queues**
    Le code (FIFO) possono essere implementate con liste o con strutture più efficienti come `collections.deque`.

15. **Tuples**
    Le tuple sono collezioni immutabili e ordinate, ideali per rappresentare dati costanti o record semplici.

16. **Swapping Variables**
    Python permette di scambiare rapidamente il valore di due variabili usando l’unpacking, senza variabili temporanee.

17. **Arrays**
    Il modulo `array` offre strutture più compatte e performanti delle liste quando si lavora con grandi quantità di dati numerici omogenei.

18. **Sets**
    I set sono collezioni non ordinate di elementi unici, ideali per operazioni matematiche come unioni, intersezioni e differenze.

19. **Dictionaries**
    I dizionari memorizzano coppie chiave-valore e rappresentano una delle strutture dati più potenti e flessibili del linguaggio.

20. **Dictionary Comprehensions**
    Come le list comprehension, consentono di creare dizionari in modo sintetico ed espressivo.

21. **Generator Expressions**
    Simili alle list comprehension, ma restituiscono generatori, utili per lavorare con grandi quantità di dati senza caricarli tutti in memoria.

22. **Unpacking Operator**
    Gli operatori `*` e `**` permettono di espandere sequenze e dizionari, utili per funzioni, merging e costruzione dinamica di collezioni.

23. **Exercise**
    Una serie di esercizi finali permetterà di consolidare tutte le tecniche e le strutture dati presentate nel capitolo.
