## Metodologia Didattica

**Approccio pythonico enfatizzato:**

- Leggibilità del codice prima di tutto
- "There should be one-- and preferably only one --obvious way to do it"
- Uso idiomatico delle features Python
- Comprehensions invece di loop quando appropriato
- Duck typing e EAFP (Easier to Ask for Forgiveness than Permission)
- Context managers per gestione risorse

**Formato lezioni:**

- 30% teoria ed esempi
- 50% esercizi pratici guidati
- 20% esercizi autonomi e code review

**Strumenti:**

- Editor moderno (VS Code)
- Git per version control (cenni)
- Jupyter Notebook per sperimentazione
- Piattaforma online per esercizi (opzionale)
