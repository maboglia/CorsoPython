# üì¶ Memorizzare dati nelle variabili

In questo esempio assegniamo valori di diversi tipi (stringa, numero, booleano) a variabili che rappresentano un prodotto del supermercato:

```python
# Storing different types of data in variables
item_name = "Banana"
item_price = 0.50
item_quantity = 100
in_stock = True

print("Item:", item_name)
print("Price: $", item_price)
print("Quantity:", item_quantity)
print("In stock:", in_stock)
```

Questo codice dimostra quanto sia semplice:

* memorizzare informazioni,
* accedere ai valori tramite il nome della variabile,
* visualizzarli in output.

Ogni variabile utilizza un tipo di dato adatto al valore che rappresenta:

| Variabile       | Tipo     | Significato          |
| --------------- | -------- | -------------------- |
| `item_name`     | stringa  | Nome del prodotto    |
| `item_price`    | float    | Prezzo               |
| `item_quantity` | intero   | Quantit√† disponibile |
| `in_stock`      | booleano | Disponibilit√†        |
