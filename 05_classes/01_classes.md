## 1) Classes (Classi)

### 📌 Cos’è una classe

Una **classe** in Python è un *modello* (template) che descrive **come devono essere fatti gli oggetti**: quali dati contengono (**attributi**) e quali azioni possono fare (**metodi**).

Un oggetto creato da una classe si chiama **istanza**.

---

### 📌 A cosa serve

Le classi servono per:

* rappresentare entità reali o logiche (Studente, Prodotto, ContoBancario…)
* organizzare codice complesso
* riusare e mantenere più facilmente il software

---

### 📌 Esempio semplice

```python
class Studente:
    pass
```

Qui `Studente` è una classe vuota (per ora non fa nulla).

---

### 📌 Creare un oggetto (istanza)

```python
s1 = Studente()
s2 = Studente()
```

`s1` e `s2` sono **due oggetti distinti**, entrambi di tipo `Studente`.

---

### 📌 Concetti chiave

* **Classe** = definizione
* **Oggetto/Istanza** = elemento creato dalla classe
* Ogni oggetto ha una sua identità e può contenere dati diversi
